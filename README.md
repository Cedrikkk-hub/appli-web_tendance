# Application Shiny – Analyse des Achats

Cette application Shiny permet d’explorer et d’analyser des données d’achats selon plusieurs critères : période, site, indicateur (nombre et montant d'achat) et département.

Elle propose trois visualisations principales :

- **Graphique d’évolution mensuelle** des indicateurs sélectionnés  
- **Tableau récapitulatif par site** (nombre d’achats et montants)  
- **Carte interactive** affichant la répartition des ventes par département  

Les données sont filtrées dynamiquement via l’interface utilisateur, et les résultats se mettent à jour automatiquement.

Le projet utilise les packages : **shiny**, **tidyverse**, **lubridate**, et **leaflet**.
